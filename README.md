# springcloud-archetype
a spring cloud microservice archetype

# How To Use
1. mvn install
2. idea `new project` -> `maven` -> `add Archetype...` input  
    `com.zqkh`  
    `springcloud-archetype`  
    `1.0-SNAPSHOT`  
3. input GroupId and ArtifactId
4. Properties add  
    package=`YourPackageName` （根包名）  
    archetypeCatalog=`local` （走本地优化速度）
